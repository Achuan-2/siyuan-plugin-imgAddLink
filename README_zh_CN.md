## 插件介绍

思源笔记目前并不支持图片添加超链接

本插件可以对图片同一行放在后面的超链接进行隐藏，并在图片右侧添加一个带超链接的图标，模拟出实现图片支持超链接的效果

## 注意事项

- 插件使用后，如果对带有链接的图片的段落进行编辑，关闭插件后链接会不显示，只有再次开启插件才会显示。如果介意，请不要使用！
